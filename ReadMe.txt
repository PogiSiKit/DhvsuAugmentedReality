(SETTING UP NODE) If you don't have the modules or the modules are not working
1. Go to [Node.js website](https://nodejs.org/en)
2. Download and install Node.js (LTS)
3. Open "Visual Studio"
4. Navigate to the folder "Augmented Reality"
5. Open Terminal
6. Type `npm install`
Note: If Node.js is already installed, skip to step 3

(START SERVER) To start the server
1. Open "Visual Studio"
2. Load the folder "Augmented Reality"
3. Open Terminal
4. Type `npm start`
5. The terminal will output the URL for the website
Note: If you make any changes or encounter errors when launching the server, please PM me
